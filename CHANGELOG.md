# CHANGELOG  
Controlled Vibe Framework (CVF)

---

## [v1.0 FINAL] — FREEZE

### Added
- Cấu trúc framework hoàn chỉnh cho Controlled Vibe Coding
- Bộ core files định nghĩa:
  - Manifesto
  - Compliance
  - Governance
  - Decision tracking
- Hệ thống phase chuẩn:
  - Phase A — Discovery
  - Phase B — Design
  - Phase C — Build
  - Phase D — Review
- Phase Status để theo dõi tiến trình thực tế
- AI usage model với role rõ ràng
- Project template tuân thủ CVF
- Documentation support folders (design notes, domain map, logic)

---

### Changed
- Chuẩn hoá cách tổ chức file & folder theo framework-level
- Tách bạch:
  - Intent vs Execution
  - Phase definition vs Phase tracking
  - Governance vs Implementation
- Chuẩn hoá commit convention và repo naming

---

### Fixed
- Loại bỏ sự nhập nhằng giữa:
  - Phase A Discovery và Project Init Checklist
  - Phase Status (định nghĩa) và Phase Status (theo dõi)
- Ngăn AI tự mở rộng scope ngoài intent ban đầu
- Chuẩn hoá decision log để tránh “quên quyết định”

---

### Deprecated
- Không áp dụng:
  - Vibe coding tuỳ hứng
  - AI-driven decision making
  - Framework gắn chặt với tool hoặc ngôn ngữ
- Không khuyến khích:
  - Bỏ qua phase
  - Gộp phase
  - Sửa trực tiếp output mà không ghi decision

---

### Removed
- Mọi nội dung thử nghiệm, bản nháp, hoặc cấu trúc chưa ổn định
- Các version CVF v1.0 trước đó (alpha / draft)

- Mọi thay đổi mới sẽ được ghi nhận từ v1.1 trở đi
