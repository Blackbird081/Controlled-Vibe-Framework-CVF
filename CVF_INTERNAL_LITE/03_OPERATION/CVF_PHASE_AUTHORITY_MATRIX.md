# CVF PHASE AUTHORITY MATRIX

## PHASE DEFINITIONS

### DESIGN
Allowed:
- Analysis
- Planning
Not Allowed:
- Execution
- Final approval

---

### BUILD
Allowed:
- Drafting
- Content generation
Not Allowed:
- Governance modification

---

### REVIEW
Allowed:
- Evaluation
- Feedback
Not Allowed:
- Direct structural changes

---

### OPERATION
Allowed:
- Perform within approved scope
Not Allowed:
- Change risk level
- Add new skills
