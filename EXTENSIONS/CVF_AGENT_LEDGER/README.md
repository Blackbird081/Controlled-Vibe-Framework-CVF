# CVF Agent Ledger

`CVF_AGENT_LEDGER` is the approved `B* Merge 5` physical merge package.

It merges the current-cycle learning pair:

- `CVF_ECO_v3.0_TASK_MARKETPLACE`
- `CVF_ECO_v3.1_REPUTATION`

Current-cycle execution class:

- `physical merge`
- compatibility wrappers remain in both original packages for one transition cycle

Internal structure is kept subsystem-oriented:

- `src/task-marketplace/`
- `src/reputation/`

This preserves readability while giving CVF one canonical learning-plane runtime package.
