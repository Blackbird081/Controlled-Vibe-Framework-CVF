# UAT Rubric - CVF Toolkit

This rubric defines minimum criteria for skill certification.

## R1 (Low Risk) Skills
- Functional correctness verified
- No hallucinated data
- Proper source citation (if applicable)

## R2 (Medium Risk) Skills
- All R1 criteria
- Deterministic outputs within tolerance range
- Financial calculations validated manually

## R3 (High Risk) Skills
- All R2 criteria
- Independent reviewer approval
- Output includes disclaimer (if decision-support)
- Audit log verified

## R4 (Critical Risk) Skills
- All R3 criteria
- Executive sign-off required
- Freeze applied post-certification
- Version tagged and locked

Failing any requirement blocks certification.
