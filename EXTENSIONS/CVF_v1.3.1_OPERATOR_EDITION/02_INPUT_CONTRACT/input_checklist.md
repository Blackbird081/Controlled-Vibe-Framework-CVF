Operator Input Checklist

Dùng checklist này trước khi giao execution cho AI.

Checklist bắt buộc (Fail nếu thiếu)

 Objective rõ ràng, có thể kiểm tra

 Scope được khóa

 Output Contract đầy đủ

 Constraints được nêu rõ

Checklist loại trừ (Fail nếu có)

 Có mô tả cách làm

 Có gợi ý chiến lược

 Có yêu cầu “tối ưu”, “sáng tạo”, “linh hoạt”

 Có hội thoại / ngôn ngữ cảm tính

Tự kiểm tra nhanh (30 giây)

Nếu AI làm sai, tôi có thể nói:

“AI vi phạm input contract”

mà **không cần giải thích thêm không?

Nếu có → Input hợp lệ

Nếu không → Input sai

Quy tắc cuối

Input càng ngắn, trách nhiệm AI càng cao.

Nếu bạn thấy cần “viết cho kỹ để AI hiểu”
→ bạn đang viết prompt, không phải input contract.

Kết thúc Input Contract.
