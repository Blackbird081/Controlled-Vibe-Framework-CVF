# OPERATOR CHECKLISTS – CVF v1.4.2

Thư mục này cung cấp **checklist thao tác** cho operator ở từng giai đoạn làm việc với CVF.

Mục tiêu:
- Giảm sai sót do thói quen cá nhân
- Chuẩn hóa cách “dùng CVF đúng”
- Cho phép tự kiểm tra trước khi escalation

Nguyên tắc:
- Checklist ≠ hướng dẫn kỹ thuật
- Checklist chỉ xác nhận: ĐÃ LÀM / CHƯA LÀM / KHÔNG ÁP DỤNG
- Không checklist nào được override Execution Rules

Cách dùng:
1. Đọc checklist tương ứng với phase
2. Tick thủ công (mental hoặc tool)
3. Nếu fail ≥ 1 critical item → dừng và escalation
