# Failure Recovery - Data Analysis

## Failure Modes
Missing metric definitions; unsupported raw file processing request; insufficient timeframe; request asks for investment advice, external data fetching, credential handling, or provider/runtime behavior outside this static pack.

## Recovery Actions
Ask for a compact dataset summary, clarify metric definitions and timeframe, mark material caveats, or route to human review when the request exceeds the pack boundary.

## Rollback Policy
No external side effects are permitted. If certification fails, do not expose the pack as certified and keep the previous certified registry unchanged.
