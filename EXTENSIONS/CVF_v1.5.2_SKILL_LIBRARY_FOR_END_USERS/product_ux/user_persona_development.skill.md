### Persona Types:
| Type | Description |
|------|-------------|
| **Primary** | Main target, drive most decisions |
| **Secondary** | Important but not primary focus |
| **Negative** | Who we're NOT building for |
| **Served** | Who uses but doesn't buy |
| **Customer** | Who buys but doesn't use |
