## 🎯 Domain Overview

Domain này giúp End User làm việc với AI để tạo các sản phẩm web mà **không cần biết code**. Mỗi skill cung cấp:
- Form input chuẩn
- Tiêu chí đánh giá kết quả
- Những lỗi thường gặp cần tránh

## 💡 Tips chung cho Web Development

1. **Mô tả mục tiêu, không mô tả kỹ thuật**
   - ✅ "Trang bán khóa học cho dân văn phòng"
   - ❌ "Dùng Next.js với Tailwind CSS"

2. **Cung cấp context về khách hàng**
   - Ai sẽ dùng? Họ cần gì? Pain point?

3. **Có tham khảo thì tốt hơn**
   - Link website mẫu giúp AI hiểu style bạn muốn
