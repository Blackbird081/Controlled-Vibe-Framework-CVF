# CVF Assessments

Đây là nơi lưu:

- independent assessment
- audit assessment
- reassessment
- post-fix / pre-fix assessment
- quality evaluation snapshots

Đặc điểm:

- có thời điểm, có phạm vi, có verdict
- thường gắn với evidence, trace, hoặc retest batch

Không đặt vào đây:

- roadmap khắc phục
- review archive theo module
- baseline frozen records
