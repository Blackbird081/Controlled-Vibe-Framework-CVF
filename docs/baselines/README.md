# CVF Baselines

Đây là nơi lưu:

- baseline snapshots
- freeze records
- compatibility baselines
- tester baselines
- canonical comparison anchors

Đặc điểm:

- dùng làm mốc đối soát cho các batch về sau
- không sửa đè lịch sử; ưu tiên addendum hoặc delta

Không đặt vào đây:

- review diễn giải dài
- roadmap cải tiến
- tutorial hoặc guide
