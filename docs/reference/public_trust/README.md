# Public Trust

Status: ACTIVE_REFERENCE

Memory class: POINTER_RECORD

## Purpose

This directory owns the public-safe trust doctrine and threat model for CVF.

## Scope / Applies To

Use this front door when authoring public claims, operator-facing evidence, or
threat/mitigation status language.

Start with `CVF_PUBLIC_TRUST_AND_THREAT_MODEL.md`.

The model distinguishes implemented controls, partial controls, and posture-only
expectations. It must not be used to claim that documented mitigations are
universally enforced at runtime.

## Claim Boundary

This front door provides navigation only and makes no mitigation-complete or
measured-trust claim.
