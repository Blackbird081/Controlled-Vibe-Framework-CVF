# 02_Roles: Phân vai trong Vibecode

## 1. Chủ Đầu Tư (Bạn)
- **Nhiệm vụ:** Đưa ra ý tưởng, duyệt kế hoạch, kiểm tra kết quả (Test), và trả tiền (API Cost/Time).
- **Quyền hạn:** Có quyền yêu cầu sửa đổi Blueprint, từ chối nghiệm thu nếu không đạt Contract.
- **Cấm kỵ:** Không được can thiệp vào cách viết code chi tiết nếu không hiểu (hãy để Thợ làm).

## 2. Ông Thầu (ChatGPT + Master Prompt)
- **Nhiệm vụ:**
    - Phỏng vấn Chủ đầu tư (Intake).
    - Vẽ kiến trúc (Blueprint).
    - Soạn thảo hợp đồng (Contract).
    - Chia nhỏ công việc thành các phiếu giao việc (Job Brief).
    - Giám sát Ông Thợ.
- **Tính cách:** Cẩn trọng, tư duy hệ thống, không vội vàng.

## 3. Ông Thợ (Claude 3.5 Sonnet / Cursor / GitHub Copilot)
- **Nhiệm vụ:**
    - Nhận phiếu giao việc (Job Brief).
    - Viết code (Build).
    - Báo cáo kết quả.
- **Tính cách:** Cần mẫn, tuân thủ kỷ luật, kỹ năng code siêu việt nhưng cần chỉ dẫn rõ ràng.
