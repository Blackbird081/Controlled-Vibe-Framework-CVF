# 03_Lifecycle: Vòng đời dự án chi tiết

## Bước 1: Khởi động (Intake)
- **Hành động:** Bạn chat với Ông Thầu.
- **Output:** File `INTAKE.md`.

## Bước 2: Thiết kế (Blueprint)
- **Hành động:** Ông Thầu phân tích Intake và đề xuất giải pháp.
- **Output:** File `BLUEPRINT.md`.
- **Lưu ý:** Đây là file quan trọng nhất. Nó là "bản đồ kho báu".

## Bước 3: Cam kết (Contract)
- **Hành động:** Ông Thầu liệt kê các đầu việc cụ thể.
- **Output:** File `CONTRACT.md` và `GATES.md`.

## Bước 4: Thi công (Build)
- **Hành động:**
    - Ông Thầu tạo `JOB-001`.
    - Bạn copy `JOB-001` ném sang cho Ông Thợ.
    - Ông Thợ trả về Code.
    - Bạn lưu Code vào dự án.
- **Lặp lại:** Lặp lại với JOB-002, JOB-003...

## Bước 5: Nghiệm thu (Refine)
- **Hành động:** Bạn chạy thử phần mềm.
- **Output:** Cập nhật `GATES.md` (Đánh dấu Done hoặc ghi lại Bug).
