# Quy ước đặt tên (Naming Conventions)

## Thư mục dự án
- Luôn có folder `.vibecode/` ở root để chứa tài liệu thiết kế.

## Mã công việc (Job ID)
- `JOB-001-Setup`: Cài đặt môi trường.
- `JOB-002-Database`: Thiết kế DB.
- `JOB-010-UI-Login`: Làm giao diện Login.

## Versioning
- Blueprint v1.0, v1.1...
- Luôn backup file cũ trước khi update lớn.
