# Tài Liệu Tham Khảo

Thư mục này chứa tài liệu nâng cao cho người muốn tìm hiểu sâu.

**Bạn KHÔNG cần đọc gì ở đây để sử dụng Vibecode Kit.**

Chỉ cần dùng các file .txt ở thư mục gốc là đủ.

## Nội dung

- `00_CORE/` - Triết lý và nguyên tắc
- `10_PROMPTS/` - Prompt templates gốc
- `20_TEMPLATES/` - Mẫu documents
- `30_EXAMPLES/` - Ví dụ tham khảo
- `40_TOOLS/` - Công cụ hỗ trợ
- `skills/` - Guidelines UI/UX chi tiết
- `legacy/` - Phiên bản cũ (MOTHER-PROMPT)
